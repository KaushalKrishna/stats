# UPSC Statistics Optional Study System

This folder is a Markdown-only study system for UPSC Statistics optional.

It uses `Syllabus CSE statistics.pdf` as the syllabus authority and treats `pyq.csv` only as a pattern signal. The aim is not to maintain a dataset. The aim is to study the right question families, write better answers, and practise realistic mixed mocks.

## How To Use

1. Read the [Master Guide](UPSC_Statistics_Master_Guide.md).
2. Use the `MUST COVER` lists to set priority.
3. Solve question-bank families before full mocks.
4. Write mocks in exam format.
5. Use the post-test review grid at the end of each mock.
