# Question Bank: Official Statistics

Use this file by family. Do not solve it top-to-bottom in one sitting. Pick a family, learn its trigger, then write 3-5 answers in exam format.

## MUST COVER: Official statistical system

**Recognition trigger:** India data system.

**Standard framework:** Agency, data, publication, limitation..

**Practice questions**
